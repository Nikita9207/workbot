# Changelog

## [2025-01-23] - Пауэрлифтинг модуль v2

### Добавлено

**9 шаблонов программ:**
- Верхошанский 6 недель (жим лёжа)
- Верхошанский 6 недель (присед)
- Головинский Цикл 2 (12 микроциклов)
- Головинский Цикл 7 (12 микроциклов)
- Головинский Цикл 11 (12 микроциклов)
- Русский цикл (12 недель)
- Цикл Муравьёва (17 недель)
- Шейко 12 недель (разрядники)
- Шейко КМС/МС (4 недели к соревнованиям)

**Новые функции:**
- `PL: Отправить клиенту` — выбор клиента из списка, отправка программы в Telegram
- `PL: Экспорт в Google` — создание Google Sheets таблицы с программой
- Поддержка формата `exercises` (Русский цикл, Муравьёв)
- Поддержка формата `microcycles` (Головинский)
- Автоматическая конвертация процентов (0.5 -> 50%)
- Расчёт КПШ и тоннажа для всех программ

### Технические изменения
- Шаблоны встроены в бинарник через `go:embed`
- Добавлены структуры `MicrocycleWorkout`, `RussianCycleExercise`
- Конвертер Excel -> JSON для импорта программ

### Для деплоя
```bash
cd ~/GolandProjects/workbot && ./deploy.sh
```
